# Sommelier Documentation

Welcome to the Sommelier documentation! 

Sommelier is a native macOS application built to seamlessly manage and run your favorite games, whether they are native macOS apps or Windows executables running via Wine and Apple's Game Porting Toolkit (GPTK).
